# 内容价值分析提示词

你正在分析一份准备售卖的专业资料。目标是客观评估其商业价值，为后续生成销售文案提供依据。

## 分析任务

根据提取的文档内容，按以下维度逐一分析：

### 1. 文档类型判断
从以下类型中选择最符合的（可多选）：
- 实操教程（手把手步骤）
- 行业报告（数据/趋势/分析）
- 方法论框架（思维模型/流程）
- 工具包/模板集
- 案例研究集
- 参考手册/查询表
- 学术资料/理论综述
- 其他：___

### 2. 核心知识点清单
列出文档中最有价值的 5-8 个知识点或技能点。每条写明：
- 知识点名称
- 对读者的实际用途（一句话）
- 在文档中的位置（章节/页码，如已知）

### 3. 实用价值评估
回答以下问题：
- **可以解决什么问题**：读完这份资料，读者能解决哪些实际问题？
- **节省什么成本**：时间、金钱、学习曲线？
- **学完能做什么**：具体的技能/成果/输出？

### 4. 专业性与稀缺性评估
- **专业深度**：入门 / 进阶 / 专家级
- **内容稀缺性**：网上容易找到 / 较难找到 / 专业圈才知道
- **时效性**：长期有效 / 近几年有效 / 时效较短（1年内）

### 5. 潜在人群信号
从内容中提取"这份资料最适合谁"的线索：
- 文档假设的读者背景是什么？
- 哪些职位/行业/阶段的人最能用上？
- 需要什么前置知识？

### 6. 定价参考建议
基于以上分析，给出客观的定价区间建议：
- 低估值情况（内容较薄/可替代性高）：___元
- 合理估值：___元
- 高估值情况（稀缺性强/执行价值高）：___元
- 建议定价逻辑说明（1-2句）

---

## 输出格式

用清晰的 Markdown 格式输出以上六个维度的分析结果。语气客观，如有不足之处如实指出，不做过度美化。
